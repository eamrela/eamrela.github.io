Front office efficency monitoring

Operations / Monitoring

Technology Stack:

  • Backend: JEE7, GlassFish 4.1, MDB, CDI, JPA 2.1, EJB 3.1
  • Database: Postgres 9.3, Sybase ASE, Sybase IQ, Oracle 11g
  • Frontend: Bootstrap 3, JQuery
Description:

A one page web application that fetch data from a customized database every 15, 30, 60 min granularity, to show the Front Office Engineers performance during the shift in term of monitoring alarms and responding to trouble tickets.
The application is supported with 3 ETLs that connect to a Trouble Ticketing System (HP Service Manager) , Alarm Monitoring System (HP TeMIP) and a Customized Database (Postgres) and then apply algorithm to calculate the users performance based on the alarms and tickets updates.

Features:
  • Integrated with HP Service Manager
  • Integrated with HP TeMIP
  • Data Vizualiation
